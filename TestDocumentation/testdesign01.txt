P01.TD01.01

Проект теста по проверке требования:
Программа должна вставлять из буфера обмена произвольный фрагмент текста без элементов форматирования.

Дата: 03.10.2022.

Разработчик: Габидуллин.

Объект тестирования:
1. Стандартное приложение OC Windows Блокнот
Текстовый редактор, который не поддерживает элементы форматирования.

Цедь теста:
Провека возможности вставить в текст документа фрагмента равный максимальному обьему буфера, но не превышающий размер
свободной оперативной памяти, которая выделена ОС приложению Блокнот.

Методика тестирования:
1. Опредеоение предельного азмера буфера.
2. Определение свободной оперативной памяти, которая доступна приложению Блокнот.
3. Из данных размеров выбирается минимальная.
4. Подготавливаем файл, содержащий фрагмент текста данного размера.
5. Текст из файла загружается в буфер обмена.
6. Содержимое буфера вставляется в текст документа.

Критерии успешности теста:
1. В документ вставляется весь текст из буфера: контроль осуществим по размеру полученного файла.
2. Текст вставляется с позиции, указанной пользователем.
3. Исходный текст документа сохраняется: нет исчезновений, нет искажений исходного текста.
